Langkah-langkah Singkat Instalasi MySQL

  1. Download installer MySQL dari situs resmi: pilih MySQL Community Server sesuai OS kamu (Windows, Mac, Linux).
  2. Jalankan installer — pilih opsi “Developer Default” atau “Server only” sesuai kebutuhan.
  3. Atur konfigurasi — buat password untuk user root, dan biarkan port default 3306.
  4. Selesai instalasi — buka MySQL Workbench atau gunakan CLI: mysql -u root -p
  5. Verifikasi — coba buat database baru: 
    CREATE DATABASE coba_db;
SHOW DATABASES;
